# sokoban

A clone of the classical japanese warehouse game.

## Playing the game

The game can be played with a keyboard, using arrows to move.

Your objective is to fill all targets with a box. Whenever you complete a
level, you automatically go to the next level. There are 60 levels in total.

The game is unforgiving on mistakes. Plan your moves accordingly or you might
have to restart levels a lot.

If you click in the gear on the top-right corner, you get to the options menu.
In there, you can enable/disable audio, select a previously played level, or
reset the level you are currently playing.
